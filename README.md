# Monte-Carlo-Planer

<https://simonfahrig.github.io/project_management/>

## Überblick

Webbasierte SaaS-Anwendung für Projektmanager, Scrum Master, Risikomanager und Product Owner

Ein intuitives Web-Tool, das durch statistische Monte-Carlo-Simulationen das realistischste Projektenddatum ermittelt.
Es verwandelt vage Schätzungen in präzise, datenbasierte Wahrscheinlichkeiten für eine sichere Projektplanung.

## Hauptfunktionen

### Drei-Punkt-Schätzung

Schnelle Eingabe von Best-Case, Most-Likely-Case und Worst-Case für einzelne Projektphasen oder Arbeitspakete.

### Risiko-Simulation

Durchführung von tausenden automatischen Simulationsdurchläufen per Knopfdruck zur Berechnung aller potenziellen Projektausgänge.

### Wahrscheinlichkeits-Diagramm

Visuelle Darstellung der Ergebnisse (z. B. „Das Projekt ist mit einer Wahrscheinlichkeit von 85% am 15. November fertig“).
